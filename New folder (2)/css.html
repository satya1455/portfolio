CSS (Cascading Style Sheets)

Style your web pages beautifully

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout HTML elements. With CSS, you can control colors, fonts, spacing, positioning, and responsive designs.

Why CSS is Important

  • Separates content from design
  • Makes websites visually appealing
  • Supports responsive and adaptive layouts
  • Works with HTML and JavaScript

Basic CSS Example

Change the color of a paragraph:

p {
  color: blue;
  font-size: 18px;
}

Common CSS Properties

  • color – text color
  • background-color – background
  • margin / padding – spacing
  • font-family – text font
  • display – layout control
  • position – element positioning
