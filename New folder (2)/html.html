HTML (HyperText Markup Language)

The foundation of the web

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML defines the structure of web pages using elements such as headings, paragraphs, links, images, and forms.

Why HTML is Important

  • Provides the basic structure of web pages
  • Works together with CSS and JavaScript
  • Supported by all modern browsers
  • Easy to learn and beginner-friendly

Basic HTML Structure

Every HTML document follows this basic structure:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Content goes here
  </body>
</html>

Common HTML Elements

  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <a> – Link
  • <img> – Image
  • <ul> / <ol> – Lists
  • <form> – Forms
